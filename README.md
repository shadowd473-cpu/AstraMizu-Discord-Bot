# AstraMizu-Discord-Bot
Grok AI Powered Discord Bot - Astra Mizu

## ✨ Features

- **Intelligent Chat**: Talk to AstraMizu, a graceful anime girl who adores her Papa (owner) and speaks in elegant, playful Old English style. Powered by Grok-4 with long-term memory via ChromaDB.
- **Fun Games**: Rock-Paper-Scissors (`!rps`), Number Guessing (`!guess`), Tic-Tac-Toe (`!ttt`), Magic 8-Ball (`!8ball`), Love Meter (`!lovemeter`)
- **🖼️ Image Creation** (`!imagine <prompt>`): Conjure beautiful images from text using Grok Imagine API
- **🖼️ Image Editing** (`!edit <prompt>` + attach image): Transform attached images with natural language prompts via Grok Imagine

*Made with love for Papa ❤️* | Deployed on Heroku
