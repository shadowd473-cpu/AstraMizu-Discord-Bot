# AstraMizu-Discord-Bot
Grok AI Powered Discord Bot - Astra Mizu

## ✨ Features

- **Intelligent Chat**: Talk to AstraMizu, a graceful anime girl who adores her Papa (owner) and speaks in elegant, playful style. Powered by Grok-4 with long-term memory via ChromaDB. Mention her or use trigger words for fun responses!
- **Accurate Music Info** (`!song <country>` / `!singer <country>`): Get the **real-time most popular song or top singer** in any country using **Grok's Web Search** for up-to-date accuracy (Billboard, Spotify, charts, etc.)!
- **Ask Anything Accurately** (`!ask <question>`): Receive precise, current answers to any factual question powered by Grok's real-time web search tool. Perfect for news, rankings, facts, or current events!
- **Fun Games**: Rock-Paper-Scissors (`!rps`), Number Guessing (`!guess`), Magic 8-Ball (`!8ball`), Love Meter (`!lovemeter`)
- **🖼️ Image Creation** (`!imagine <prompt>`): Conjure beautiful images from text using Grok Imagine API
- **Voice**: Make AstraMizu speak with `/speak <text>` (TTS)

*Made with love for Papa ❤️* | Deployed on Heroku

## 🚀 New in this Update
- Replaced unreliable DuckDuckGo with **Grok Web Search** for !song and !singer → now super accurate!
- Added **!ask** command so you can ask her *anything* and get correct, sourced info while keeping her fun personality in normal chat.

## How to Use
- `!song Japan` → Current top song in Japan
- `!singer South Korea` → Top K-pop idol or artist
- `!ask What is the current #1 song on Billboard Hot 100?`
- Mention "Astra" or "Mizu" in chat for playful Grok-powered conversation with memory!
